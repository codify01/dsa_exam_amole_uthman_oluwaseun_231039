What is lists/arrays

Lists or array is a type of data type that is used to stored multiple values. they are useful when managing series of items or list

Array methods

  • push() - adds an item to the end of an array
  • pop() - removes the last item from an array
  • shift() - removes the first item from an array
  • unshift() - adds an item to the beginning of an array
  • splice() - adds or removes items from a specific index in an array. This was used while creating mini e-commerce app in level two
